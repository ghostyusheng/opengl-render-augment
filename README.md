[Paper Title]: Real-Time Pencil Rendering

[Author(s)]: Hyunjun Lee

Published in Conference, 05 June 2006

DOI: https://dl.acm.org/doi/pdf/10.1145/1124728.1124735

The paper explores real-time non-photorealistic rendering, particularly simulating a hand-drawn sketch effect in OpenGL.
 The key techniques include:
Edge detection using a Sobel filter to extract contours,


Hatching shaders to simulate pencil strokes,


And tone mapping to imitate the texture and feel of a sketch.


These methods are all about achieving a natural pencil-drawing look without sacrificing performanceâ€”useful in games and stylized animations.
